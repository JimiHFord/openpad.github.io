{openpad} Multiplier Games Made Easy

What is openpad?


Openpad is a c++ library that allows you to make your game playable using a smartphone as a controller.

Simple



Creating multiplayer games using openpad is simple and easy. Implement a few callback methods, run the server, and you're good to go.


Powerful


Openpad gives you complete control over the controller layout. Player controls are completely customizable, and you can even change the controls during the game.
